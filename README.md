# SecretVault

A truly invisible Android vault app — no launcher icon, no traces. Access your hidden apps only through a secret dialer code.

## Features

- **No Launcher Icon** — The app has no icon in your app drawer. It doesn't exist.
- **Secret Dialer Access** — Dial `*#*#6789#*#*` to open the vault.
- **PIN Protected** — 4+ digit PIN to prevent unauthorized access.
- **Hide Any App** — Select any installed app to hide it from the launcher.
- **Easy Launch** — Open hidden apps directly from the vault grid.
- **Lightweight** — Only ~17KB. No permissions required.

## How to Use

1. Install the APK
2. Open via dialer: `*#*#6789#*#*`
3. Set your PIN on first launch
4. Add apps to hide using the + button
5. (Optional) Use "Hide Icon" to remove the launcher component
