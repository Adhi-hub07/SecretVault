✓ Features

Calculator Disguise

Appears as a normal Calculator app with a working calculator. Nobody suspects a thing.

Secret Code: 2007

Type 2007 in the calculator and press = — the vault opens. Any other calculation works normally.

PIN Protected

Set a 4+ digit PIN on first use. Required every time you access the vault.

Hide Any App

Select apps from the list to hide. They disappear from launcher, search, everywhere.

Launch Hidden Apps

Tap any hidden app in the vault to launch it. It briefly reappears, then vanishes after 3 seconds.

Dialer Code

Also works by dialing *#*#6789#*#* in your phone app — skips the calculator.

Root Mode

Auto-detects root. Uses su -c "pm hide" to make apps invisible from Settings and Play Store too. Green "root" badge shown in vault. Falls back to non-root hiding if no root.

⚙ How to Use

1

Install the APK

Download and install it. The app shows up as "Calculator" with a shield icon.

2

Open the Calculator

Tap the Calculator icon. It works as a real calculator — add, subtract, multiply, divide.

3

Unlock the Vault

Type 2007 using the calculator buttons, then press =. The PIN screen appears.

4

Set Your PIN (First Time)

Enter a 4+ digit PIN and confirm. This PIN will be required every time you open the vault.

5

Add Apps to Hide

Inside the vault, tap "+ Add Apps". Browse the list and tap any app to hide it. With root: completely vanishes from Settings & Play Store too. Without root: vanishes from launcher & search. To unhide: long-press the app in the vault.

6

Use Hidden Apps

Tap any hidden app from the vault grid to launch it. It re-appears for 3 seconds while you use it, then vanishes again. Long-press to permanently unhide and restore visibility.

⚠ Root vs Non-Root

With Root

Hidden apps completely vanish from the system — no icon in launcher, no trace in Settings → Apps, not visible in Play Store. Uses pm hide via su. Green "root" badge shown on hidden apps in vault.

Without Root

Hidden apps disappear from launcher, app drawer, and system search. Still visible in Settings → Apps and Play Store. Android doesn't allow non-root apps to hide from system settings. No system apps can be hidden.
