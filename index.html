Features

🔢

Calculator Disguise

Looks and works like a real calculator. Hidden vault is triggered by long-pressing the display.

🔒

PIN Protected

4+ digit PIN prevents unauthorized access to the vault and hidden apps.

📱

Hide Any App

Select any installed app to hide — it disappears from the launcher and search instantly.

🚀

Quick Launch

Tap any hidden app from the vault grid to launch it. It reappears briefly, then vanishes again.

📞

Secret Dialer Code

Also accessible via *#*#6789#*#* in the phone dialer.

🔐

Self-Hiding

Use "Hide Icon" inside the vault to disable the icon. Only accessible via dialer code after that.

How It Works

1

Install the APK

Download and install the APK. It appears as "Calculator" with a calculator icon.

2

Use the Calculator (or Dialer Code)

Open the app — it's a fully working calculator. Or dial *#*#6789#*#* for direct access.

3

Access the Vault

Long-press the calculator display to open the PIN screen. Enter your PIN to access hidden apps.

4

Add Apps to Hide

Tap the + button to browse installed apps. Selected apps disappear from launcher and search.

5

Launch Hidden Apps

Tap any hidden app from the vault to launch it. Long-press to permanently unhide and restore its icon.
